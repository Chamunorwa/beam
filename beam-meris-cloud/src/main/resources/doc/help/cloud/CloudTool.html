Cloud Probability Processor

The BEAM Cloud Probability Processor implements the detection of clouds in a MERIS L1b product. The created output product contains one raster indicating the probability that the current pixel is a cloud pixel, and three additional flags to the already present l1_falgs. The three flags indicate pixels which are cloudy (probability > 80%), cloud free (probability < 20%) or where it is uncertain (20% < probability < 80%).
