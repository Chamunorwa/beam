Tools Menu

VISAT's Tools Menu

The following tools all generate new products from the currently selected input product(s). To view their created raster datasets after processing is complete, change to the product browser, expand the new product's nodes and double click on a band node to open an image view. Note that you really get new products which can be saved and opened again for later use.

  • Band Arithmetics
  • Map Projection
  • Orthorectification
  • Data Flip
  • Attach Pixel Geo-Coding
  • Detach Pixel Geo-Coding
  • Create Subset from View
  • Create Elevation Band
  • Create ASAR NRCS Bands
  • Create Filtered Band

VISAT Integration of Scientific Data Processors

The following scientific data processors are also provided as stand-alone applications to be run either from the command-line in batch mode or interactive mode. Of course, within VISAT they are exclusively run in interactive mode.

  • SMAC Processor
  • Smile Correction Processor
  • Cloud Probility Processor
  • Radiance-To-Reflectance Conversion Processor
  • SST Processor
  • FLH/MCI Processor
  • Level 3 Mosaic Processor
  • Level 3 Binning Processor
