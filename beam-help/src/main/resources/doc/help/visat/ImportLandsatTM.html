Import Landsat TM

The Landsat TM reader enables VISAT to import Landsat TM data products in either the Fast format (Landsat5) or in GeoTIFF (Landsat 4, 5, 7, 8) format.

More about Landsat TM characteristics can be found at http://landsat.usgs.gov/resources/project_documentation.php.
