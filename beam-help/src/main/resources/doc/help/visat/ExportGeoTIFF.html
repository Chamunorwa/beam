Export GeoTIFF

Export GeoTIFF Product

This command allows to export the data of the current product to GeoTIFF file.

GeoTIFF is an extension to the TIFF 6.0 specifiation. These image files contain georeferencing and geocoding informations. A wide range of projected coordinate systems are supported including UTM, US State Plane, and National Grids. Also GeoTIFF supports numerous projection types. For further information have a look at the following documentations:

  • GeoTIFF Specification
  • TIFF Specification

Constraints to created GeoTIFF product files

  • Band rasters are always written as 32-bit float values.
  • Visat virtual bands are not exported.
  • For a list of supported projections, see the chapter Available Map Projections.
