Edit Menu

VISAT's Edit Menu

Delete  (Control-D)

Deletes a selected object. This command currently works only for the bands of a product and therefore the delete command is only enabled if a band has been selected in the product tree view or an image view has been activated.

Delete Shape   

Deletes a geometric shape in the current image view. The command is enabled only if a transect data (geometric shape figure) was imported or drawn in the current image view.

Delete Pin   

Deletes a selected pin in the current image view. The command is enabled only a pin is selected. New pins are created using the pin tool.

Shape To ROI   

Converts a geometric shape in the current image view to a region of interest (ROI). After a shape has become a ROI, a statistical analysis can be performed on it. The command is enabled only if a shape is available (see above).

ROI To Shape   

Converts a ROI defined in the selected band into a geometric shape in the current image view. After a ROI has become a shape, statistical analysis requiring a geometric path, e.g. transect can be performed on it. The command is enabled only if a ROI shape is defined in the displayed band.

Properties...  

This menu item opens the property editor which can be used to edit the properties of a product, a tie-point grid or a (virtual) band.

Preferences...    (Alt-P)

Opens VISAT's preferences dialog which lets your preferred VISAT options.
